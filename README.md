##  React Account Manager Application

A simple React (v16+) application that allows users to register, login, view profile and logout. 


##  Features

- User Registration with full name, email & password
- Email validation using regex
- Password validation (minimum 6 characters)
- Full name validation (no numbers allowed)
- User login with credential verification
- Protected profile page (accessible only after login)
- Welcome message with userâ€™s name
- Edit profile functionality (update name)


##  Tech Stack

- React.js
- React Router DOM
- Context API
- LocalStorage (used as mock backend)
- CSS (responsive styling)
